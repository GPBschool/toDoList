Возвращаемые данные от сервера

Получить список всех дел    GET(localhost:8080/)
Возвращаемое значение масив объектов ToDo [{"id":0,"name":"111","description":"1111"},{"id":1,"name":"Дело 1","description":"Выполнить что-то"}]

Получить дело с id 1        GET(localhost:8080/1)
Возвращаемое значение объект ToDo {"id":1,"name":"Дело 1","description":"Выполнить что-то"}

Создание нового дела        POST(localhost:8080/)
                            (Content-Type: application/json
                            {"name":"Наименование дела","description":"Описание дела"})
Возвращаемое значение созданного объекта ToDo {"id":2,"name":"Наименование дела","description":"Описание дела"}

Обновление дела             PUT(localhost:8080/2)
                            (Content-Type: application/json
                            {"name":"Новое наименование дела","description":"Новое описание дела"})
Возвращаемое значение измененный объект ToDo {"id":2,"name":"Новое наименование дела","description":"Новое описание дела"}


Удаление дел по id 2         DELETE(localhost:8080/2)
